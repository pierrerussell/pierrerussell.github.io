System Architecture

This project implements Clean Architecture with Domain-Driven Design principles, ensuring clear separation of concerns and testability. Each layer has specific responsibilities and dependencies flow inward toward the domain.

public class CompanyAppService : ICompanyAppService
{
    private readonly ICompanyRepository _companyRepository;
    private readonly ICompanySearchGateway _searchGateway;

    public async Task<IEnumerable<CompanyDto>> SearchCompaniesAsync(string query)
    {
        // 1. First, check local database
        var localCompanies = await _companyRepository.SearchAsync(query);
        
        if (localCompanies.Any())
        {
            return ObjectMapper.Map<IEnumerable<CompanyDto>>(localCompanies);
        }

        // 2. If not found locally, search via FMP
        var fmpCompanies = await _searchGateway.SearchCompaniesAsync(query);
        
        // 3. Store in database for future use
        foreach (var company in fmpCompanies)
        {
            await _companyRepository.AddAsync(company);
        }
        await _companyRepository.SaveChangesAsync();

        return ObjectMapper.Map<IEnumerable<CompanyDto>>(fmpCompanies);
    }
}

🎯 Rate Limiting Strategy

FMP API has a 250 calls/day limit. Our caching strategy ensures we minimize API calls by:

  • Always checking the database first
  • Bulk fetching when possible
  • Storing all fetched data for reuse
  • Implementing exponential backoff for retries

Portfolio Optimization Engine

The portfolio optimization module implements Modern Portfolio Theory using efficient frontier calculation with support for real-world constraints.

Mathematical Foundation

Markowitz Mean-Variance Optimization

We solve the quadratic optimization problem:

Minimize: wTΣw - λ·wTμ
Subject to: Σwi = 1, 0 ≤ wi ≤ wmax

Where:

  • w = portfolio weights vector
  • Σ = covariance matrix of returns
  • μ = expected returns vector
  • λ = risk aversion parameter

F# Implementation

Portfolio Optimization Core

StockSimulation.Quant/Portfolio/Optimizer.fs

module StockSimulation.Quant.Portfolio.Optimizer

open MathNet.Numerics.LinearAlgebra

type OptimizationConstraints = {
    NoShortSelling: bool
    MaxPositionSize: float option
    MinPositionSize: float option
}

type OptimizationResult = {
    Weights: Map<string, float>
    ExpectedReturn: float
    PortfolioRisk: float
    SharpeRatio: float
}

let calculateCovariance (returns: Matrix<float>) =
    let meanReturns = returns.ColumnSums() / float returns.RowCount
    let centeredReturns = returns - DenseMatrix.ofRowVector meanReturns
    (centeredReturns.Transpose() * centeredReturns) / float (returns.RowCount - 1)

let optimizePortfolio (returns: Matrix<float>) 
                     (symbols: string list) 
                     (constraints: OptimizationConstraints) 
                     (riskAversion: float) =
    
    let covariance = calculateCovariance returns
    let meanReturns = returns.ColumnMeans()
    
    // Set up quadratic programming problem
    let Q = 2.0 * covariance  // Quadratic term
    let c = -riskAversion * meanReturns  // Linear term
    
    // Constraints setup
    let constraints = 
        [ // Sum of weights = 1
          EqualityConstraint(DenseVector.create symbols.Length 1.0, 1.0)
          
          // No short selling
          if constraints.NoShortSelling then
              yield InequalityConstraint(
                  DenseMatrix.identity symbols.Length,
                  DenseVector.create symbols.Length 0.0
              )
          
          // Max position size
          match constraints.MaxPositionSize with
          | Some maxSize ->
              yield InequalityConstraint(
                  -DenseMatrix.identity symbols.Length,
                  DenseVector.create symbols.Length (-maxSize)
              )
          | None -> ()
        ]
    
    // Solve using quadratic programming solver
    let solution = QuadraticProgramming.solve Q c constraints
    
    // Calculate portfolio metrics
    let weights = solution.X
    let portfolioReturn = meanReturns * weights
    let portfolioRisk = sqrt(weights * covariance * weights)
    let sharpeRatio = portfolioReturn / portfolioRisk * sqrt(252.0)
    
    {
        Weights = 
            symbols 
            |> List.mapi (fun i sym -> sym, weights.[i])
            |> Map.ofList
        ExpectedReturn = portfolioReturn * 252.0  // Annualized
        PortfolioRisk = portfolioRisk * sqrt(252.0)  // Annualized
        SharpeRatio = sharpeRatio
    }

Risk Analytics Engine

The risk engine implements multiple Value at Risk (VaR) methodologies and provides comprehensive risk attribution analysis.

VaR Implementation

Three VaR Methods

StockSimulation.Quant/Risk/ValueAtRisk.fs

module StockSimulation.Quant.Risk.ValueAtRisk

open System
open MathNet.Numerics.Statistics

type VaRResult = {
    HistoricalVaR: decimal
    ParametricVaR: decimal
    MonteCarloVaR: decimal
    ConditionalVaR: decimal
}

// Historical VaR - empirical quantile method
let calculateHistoricalVaR (returns: float[]) (confidence: float) =
    let alpha = 1.0 - confidence
    let sortedReturns = returns |> Array.sort
    let index = int(Math.Floor(float sortedReturns.Length * alpha))
    sortedReturns.[index]

// Parametric VaR - assumes normal distribution
let calculateParametricVaR (returns: float[]) (confidence: float) =
    let mean = Statistics.Mean(returns)
    let stdDev = Statistics.StandardDeviation(returns)
    let zScore = Normal.InvCDF(0.0, 1.0, 1.0 - confidence)
    mean + zScore * stdDev

// Monte Carlo VaR - simulation based
let calculateMonteCarloVaR (returns: float[]) 
                          (confidence: float) 
                          (numSimulations: int) =
    let mean = Statistics.Mean(returns)
    let stdDev = Statistics.StandardDeviation(returns)
    let rng = System.Random()
    
    let simulations = 
        [| for _ in 1..numSimulations do
            let z = Normal.Sample(rng, 0.0, 1.0)
            mean + stdDev * z |]
    
    calculateHistoricalVaR simulations confidence

// Conditional VaR (Expected Shortfall)
let calculateCVaR (returns: float[]) (confidence: float) =
    let varThreshold = calculateHistoricalVaR returns confidence
    returns
    |> Array.filter (fun r -> r <= varThreshold)
    |> Array.average

// Combined VaR calculation
let calculateVaR (portfolioReturns: float[]) 
                 (portfolioValue: decimal) 
                 (confidence: float) 
                 (timeHorizon: int) =
    
    // Scale returns to time horizon
    let scaledReturns = 
        portfolioReturns 
        |> Array.map (fun r -> r * sqrt(float timeHorizon))
    
    let historicalVaR = calculateHistoricalVaR scaledReturns confidence
    let parametricVaR = calculateParametricVaR scaledReturns confidence
    let monteCarloVaR = calculateMonteCarloVaR scaledReturns confidence 10000
    let cvar = calculateCVaR scaledReturns confidence
    
    {
        HistoricalVaR = portfolioValue * decimal historicalVaR
        ParametricVaR = portfolioValue * decimal parametricVaR
        MonteCarloVaR = portfolioValue * decimal monteCarloVaR
        ConditionalVaR = portfolioValue * decimal cvar
    }

Risk Attribution

Component VaR Calculation

StockSimulation.Quant/Risk/RiskAttribution.fs

module StockSimulation.Quant.Risk.RiskAttribution

// Calculate marginal VaR contribution of each position
let calculateMarginalVaR (weights: Vector<float>) 
                        (covariance: Matrix<float>) 
                        (portfolioStdDev: float) =
    (covariance * weights) / portfolioStdDev

// Calculate component VaR (contribution to total risk)
let calculateComponentVaR (weights: Vector<float>) 
                         (marginalVaR: Vector<float>) =
    weights .* marginalVaR

// Risk attribution by position
let attributeRisk (positions: Map<string, float>) 
                  (covariance: Matrix<float>) =
    let symbols = positions |> Map.toList |> List.map fst
    let weights = DenseVector.ofArray (
        symbols |> List.map (fun s -> positions.[s]) |> List.toArray
    )
    
    let portfolioVariance = weights * covariance * weights
    let portfolioStdDev = sqrt portfolioVariance
    
    let marginalVaR = calculateMarginalVaR weights covariance portfolioStdDev
    let componentVaR = calculateComponentVaR weights marginalVaR
    
    symbols
    |> List.mapi (fun i symbol ->
        symbol, {|
            Weight = weights.[i]
            MarginalVaR = marginalVaR.[i]
            ComponentVaR = componentVaR.[i]
            RiskContribution = componentVaR.[i] / portfolioStdDev
        |}
    )
    |> Map.ofList

🎯 Risk Calculation Considerations

  • Data Quality: We handle missing prices by forward-filling
  • Fat Tails: Monte Carlo VaR helps capture non-normal distributions
  • Correlation Stability: We use rolling windows for dynamic correlation
  • Computational Efficiency: Matrix operations are cached where possible

Strategy Interface

module StockSimulation.Quant.Backtesting.Strategy

type MarketData = {
    Symbol: string
    Date: DateTime
    Open: decimal
    High: decimal
    Low: decimal
    Close: decimal
    Volume: int64
}

type Signal = 
    | Buy of symbol: string * quantity: int
    | Sell of symbol: string * quantity: int
    | Hold

type StrategyState = {
    Positions: Map<string, int>
    Cash: decimal
    LastSignals: Map<string, DateTime>
}

type IStrategy =
    abstract member GenerateSignal: 
        currentData: MarketData[] -> 
        historicalData: MarketData[] -> 
        state: StrategyState -> 
        Signal list

// Moving Average Crossover Strategy
type MovingAverageCrossover(shortPeriod: int, longPeriod: int) =
    interface IStrategy with
        member this.GenerateSignal currentData historicalData state =
            currentData
            |> Array.map (fun data ->
                let history = 
                    historicalData 
                    |> Array.filter (fun h -> h.Symbol = data.Symbol)
                    |> Array.sortByDescending (fun h -> h.Date)
                    |> Array.take (max shortPeriod longPeriod)
                
                if history.Length < longPeriod then Hold
                else
                    let shortMA = 
                        history |> Array.take shortPeriod 
                        |> Array.averageBy (fun h -> float h.Close)
                    
                    let longMA = 
                        history |> Array.take longPeriod
                        |> Array.averageBy (fun h -> float h.Close)
                    
                    let currentPrice = float data.Close
                    let position = Map.tryFind data.Symbol state.Positions
                    
                    match position with
                    | None when shortMA > longMA && currentPrice > shortMA ->
                        Buy(data.Symbol, 100)
                    | Some qty when qty > 0 && shortMA < longMA ->
                        Sell(data.Symbol, qty)
                    | _ -> Hold
            )
            |> Array.toList

Backtesting Engine

Core Backtesting Logic

StockSimulation.Quant/Backtesting/Engine.fs

module StockSimulation.Quant.Backtesting.Engine

type TransactionCosts = {
    Commission: decimal
    SpreadBasisPoints: int
    MarketImpactBasisPoints: int
}

type BacktestResult = {
    TotalReturn: float
    AnnualizedReturn: float
    SharpeRatio: float
    MaxDrawdown: float
    WinRate: float
    TotalTrades: int
    TransactionCosts: decimal
    DailyReturns: float[]
}

let executeBacktest (strategy: IStrategy) 
                   (marketData: MarketData[]) 
                   (initialCapital: decimal) 
                   (costs: TransactionCosts) =
    
    let mutable state = {
        Positions = Map.empty
        Cash = initialCapital
        LastSignals = Map.empty
    }
    
    let mutable portfolioValues = [| initialCapital |]
    let mutable trades = []
    
    // Group data by date
    let dateGroups = 
        marketData 
        |> Array.groupBy (fun d -> d.Date)
        |> Array.sortBy fst
    
    // Walk through time
    for date, dayData in dateGroups do
        let historicalData = 
            marketData 
            |> Array.filter (fun d -> d.Date < date)
        
        // Generate signals
        let signals = strategy.GenerateSignal dayData historicalData state
        
        // Execute trades
        for signal in signals do
            match signal with
            | Buy(symbol, quantity) ->
                let price = 
                    dayData 
                    |> Array.find (fun d -> d.Symbol = symbol)
                    |> fun d -> d.Close
                
                let cost = calculateTransactionCost price quantity costs
                let totalCost = (price * decimal quantity) + cost
                
                if state.Cash >= totalCost then
                    state <- {
                        state with
                            Positions = 
                                state.Positions 
                                |> Map.add symbol quantity
                            Cash = state.Cash - totalCost
                    }
                    
                    trades <- {|
                        Date = date
                        Symbol = symbol
                        Side = "Buy"
                        Quantity = quantity
                        Price = price
                        Cost = cost
                    |} :: trades
            
            | Sell(symbol, quantity) ->
                let price = 
                    dayData 
                    |> Array.find (fun d -> d.Symbol = symbol)
                    |> fun d -> d.Close
                
                let cost = calculateTransactionCost price quantity costs
                let totalProceeds = (price * decimal quantity) - cost
                
                state <- {
                    state with
                        Positions = 
                            state.Positions 
                            |> Map.remove symbol
                        Cash = state.Cash + totalProceeds
                }
                
                trades <- {|
                    Date = date
                    Symbol = symbol
                    Side = "Sell"
                    Quantity = quantity
                    Price = price
                    Cost = cost
                |} :: trades
            
            | Hold -> ()
        
        // Calculate portfolio value
        let positionValue = 
            state.Positions
            |> Map.toSeq
            |> Seq.sumBy (fun (symbol, qty) ->
                let price = 
                    dayData 
                    |> Array.tryFind (fun d -> d.Symbol = symbol)
                    |> Option.map (fun d -> d.Close)
                    |> Option.defaultValue 0m
                price * decimal qty
            )
        
        let totalValue = state.Cash + positionValue
        portfolioValues <- Array.append portfolioValues [| totalValue |]
    
    // Calculate performance metrics
    calculatePerformanceMetrics portfolioValues trades initialCapital

Performance Metrics

Comprehensive Metrics Calculation

StockSimulation.Quant/Backtesting/Metrics.fs

let calculatePerformanceMetrics portfolioValues trades initialCapital =
    let returns = 
        portfolioValues
        |> Array.pairwise
        |> Array.map (fun (prev, curr) -> 
            float ((curr - prev) / prev))
    
    let totalReturn = 
        float ((Array.last portfolioValues - initialCapital) / initialCapital)
    
    let annualizedReturn = 
        let years = float returns.Length / 252.0
        Math.Pow(1.0 + totalReturn, 1.0 / years) - 1.0
    
    let sharpeRatio = 
        let avgReturn = Array.average returns
        let stdDev = 
            returns 
            |> Array.map (fun r -> Math.Pow(r - avgReturn, 2.0))
            |> Array.average
            |> sqrt
        (avgReturn * 252.0) / (stdDev * sqrt(252.0))
    
    let maxDrawdown = 
        let mutable peak = portfolioValues.[0]
        let mutable maxDD = 0m
        
        for value in portfolioValues do
            if value > peak then peak <- value
            let drawdown = (peak - value) / peak
            if drawdown > maxDD then maxDD <- drawdown
        
        float maxDD
    
    let winRate = 
        let profitableTrades = 
            trades
            |> List.filter (fun t -> t.Side = "Sell")
            |> List.filter (fun t ->
                let buyTrade = 
                    trades 
                    |> List.find (fun b -> 
                        b.Symbol = t.Symbol && 
                        b.Side = "Buy" && 
                        b.Date < t.Date)
                t.Price > buyTrade.Price
            )
            |> List.length
        
        float profitableTrades / float (List.length trades / 2)
    
    {
        TotalReturn = totalReturn
        AnnualizedReturn = annualizedReturn
        SharpeRatio = sharpeRatio
        MaxDrawdown = maxDrawdown
        WinRate = winRate
        TotalTrades = List.length trades
        TransactionCosts = 
            trades |> List.sumBy (fun t -> t.Cost)
        DailyReturns = returns
    }

StockSimulation.API/Middleware/ErrorHandlingMiddleware.cs

public class ErrorHandlingMiddleware
{
    private readonly RequestDelegate _next;
    private readonly ILogger<ErrorHandlingMiddleware> _logger;

    public async Task InvokeAsync(HttpContext context)
    {
        try
        {
            await _next(context);
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "An unhandled exception occurred");
            await HandleExceptionAsync(context, ex);
        }
    }

    private static async Task HandleExceptionAsync(
        HttpContext context, 
        Exception exception)
    {
        context.Response.ContentType = "application/json";

        var response = exception switch
        {
            ValidationException validationEx => new ErrorResponse
            {
                Message = "Validation failed",
                Details = validationEx.Errors,
                StatusCode = StatusCodes.Status400BadRequest
            },
            
            EntityNotFoundException notFoundEx => new ErrorResponse
            {
                Message = notFoundEx.Message,
                StatusCode = StatusCodes.Status404NotFound
            },
            
            ExternalServiceException serviceEx => new ErrorResponse
            {
                Message = "External service error",
                Details = serviceEx.Message,
                StatusCode = StatusCodes.Status503ServiceUnavailable
            },
            
            _ => new ErrorResponse
            {
                Message = "An error occurred processing your request",
                StatusCode = StatusCodes.Status500InternalServerError
            }
        };

        context.Response.StatusCode = response.StatusCode;
        
        await context.Response.WriteAsync(
            JsonSerializer.Serialize(response, new JsonSerializerOptions
            {
                PropertyNamingPolicy = JsonNamingPolicy.CamelCase
            }));
    }
}

Challenge 4: Data Quality

Problem

Stock data had gaps due to holidays, halts, and delistings. Simple calculations failed with missing data.

Solution

// Intelligent data cleaning
let cleanPriceData (prices: StockPrice[]) =
    prices
    |> Array.sortBy (fun p -> p.Date)
    |> Array.fold (fun acc price ->
        match acc with
        | [] -> [price]

            // Fill large gaps with interpolated data

            filled @ acc

Lessons Learned

Technical Insights

  • F# for Finance: Immutability and type safety caught many calculation errors at compile time
  • Clean Architecture Pays Off: Easy to swap data sources and add new calculations without touching core logic
  • Cache Everything: Financial calculations are expensive; aggressive caching is essential
  • Test with Real Data: Synthetic data hides edge cases that break production systems

Domain Insights

  • Simple strategies often outperform complex ones after costs
  • Risk management is more important than return optimization
  • Data quality matters more than algorithm sophistication
  • Always account for survivorship bias in historical analysis
